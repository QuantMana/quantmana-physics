## 📜 About the Project

This is not a work of fiction. It is a **scientific reduction of magic** to a natural‑science paradigm with integrated mathematical formalisation.

The model describes magic as a physiological function of organisms with a circulatory system, using differential equations, thermodynamics and biochemistry. It formalises:

- Mana neutralisation (stoichiometry 1:1:1:1).
- Spellcasting mechanics and cognitive limits of a mage.
- Critical states of the "bank" (evolution or death).
- Mechanisms behind dragons, undead and mana‑based biomes.

This is a ready‑to‑use **conceptual engine** for worldbuilding, tabletop and computer RPGs, and fantasy fiction.
